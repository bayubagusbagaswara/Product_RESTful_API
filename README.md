## Create New Product

Request :
- Method : POST
- Endpoint : `/api/products`
- Header :
    - Content-Type (consumes): application/json
    - Accept (produces): application/json
- Body :
```json
{
  "id" : "string unique",
  "name" : "string",
  "price" : "long",
  "quantity" : "integer",
  "description": "string"
}
```

Response :

```json
{
  "code": "number",
  "status": "string",
  "data": {
    "id": "string unique",
    "name": "string",
    "price": "long",
    "quantity": "integer",
    "description": "string",
    "createdAt": "date",
    "updatedAt": "date"
  }
}
```

- Endpoint : `/api/products/{id_product}`
- Header :
